# 🚀 Futuristic Developer Portfolio – Refined Project Plan Built with **Next.js 15**, **TypeScript**, **Tailwind CSS v4**, and **AI Integration** A futuristic, glowing, and professional portfolio crafted specifically for developers. Designed for **performance**, **aesthetic impact**, and **intelligent interaction** via an AI assistant — all wrapped in a neon-infused, dark-themed experience. --- ## 🔧 Initial Setup & Dependencies ### 1. ✅ Project Initialization * Base Framework: **Next.js 15 (App Router)** * Language: **TypeScript** * Styling: **Tailwind CSS v4** + **PostCSS** * Component System: **shadcn/ui** built on **Radix UI** * Markdown & Code: **MDX** + **Shiki** for syntax highlighting * Motion: **Framer Motion** * Scrolling: **Lenis** for fluid scroll behavior * AI: **Gemini API** (streaming) or compatible alternatives * Analytics: Optional **Umami** integration * Fonts: Geist Sans, JetBrains Mono (for code), Inter --- ## 🎨 Theme & UI Foundation ### 2. 🌑 Dark-First Theming * Default: **Dark Mode** * Background: #0A0A0A (Deep Space Gray) * Neon Accents: #00FFFF (Cyan Glow) & #7A00FF (Violet Pulse) * Subtle borders: rgba(255, 255, 255, 0.1) * Optional Light Mode: * Background: #F8FAFC * Accent: #0077FF * Text: #0F172A ### 3. 🧬 Design System * **Glassmorphism:** Backdrop blur panels with layered depth * **Glow Effects:** Neon pulses on hover, interactive elements * **Font Rendering:** Subpixel antialiasing for ultra-sharp type * **Animations:** * **Framer Motion** for entrance & hover animations * **Lenis** for immersive smooth scrolling * **Magnetic Cursor** effects * Scroll-triggered reveals --- ## 📁 Project Structure Overview

--- ## 🧩 Component Highlights ### Common Components * **Navbar:** Gradient glow logo, theme toggle, smooth links * **Footer:** Site-wide navigation + social icons * **ThemeProvider / ThemeSwitch:** Integrated with next-themes * **Container:** Responsive layout wrapper * **ChatBubble:** Gemini AI assistant floating chat * **OnekoCat:** Interactive cat that follows the cursor * **SectionHeading:** Neon-underline headers * **Skill:** Tooltip-based tech tags * **BackToTop:** Scroll-aware floating button ### Landing Page Sections * **Hero:** Avatar, dynamic intro, gradient text * **About:** Developer bio + interactive skill set * **Experience:** Work timeline using cards * **Projects:** Grid of featured case studies * **CTA:** Contact/engagement prompts with visuals --- ## 🛠️ Key Pages Breakdown ### 1. Homepage (/) A bold, interactive landing page with: * Floating Oneko cat * ChatBubble AI assistant * Hero section with animated avatar * Skills & tools preview * Experience timeline * Featured projects * CTA panel linking to contact and resume ### 2. Projects (/projects) * Grid layout of all showcased work * Interactive filterable cards * Dynamic detail pages powered by MDX * Rich media: screenshots, links, technology stack * Related projects navigation ### 3. Resume (/resume) * Structured PDF-style resume * Download option * Clear sectioning (Skills, Experience, Education) ### 4. Work Experience (/work-experience) * Detailed timeline using company logos, roles, and descriptions * Responsive and scroll-animated ### 5. Contact (/contact) * Clean contact form with validation * Integrated with **Bolt Database** to store submissions * Auto-response & optional alerts --- ## 🤖 Gemini AI Assistant ### Features: * Powered via **streaming API** from Gemini * Markdown-supported chat bubbles * Portfolio-specific knowledge (project summaries, experience) * Suggested prompts like: * “Tell me about your latest project” * “Where have you worked before?” * “Download your resume” * Error handling, rate limits, and loading states included --- ## 📦 Database Integration (BoltDB) ### BoltDB Use Cases: * Store **contact form submissions** * (Optional) Track **project view counts** * Setup secure API routes with form validation * Environment-secured credentials --- ## 💡 Animations & Enhancements * **Framer Motion** page and section transitions * **Lenis** scroll physics * **Magnetic Cursor** for interactive buttons/cards * **Parallax** and floating effects * **Code block highlighting** via **Shiki** * **Oneko Cat** with sprite animation and cursor following ---
